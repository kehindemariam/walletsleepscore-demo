How it works

One job: answer "is this wallet alive?" with a single number.

  1. Fetch live data from rpc.pharos.xyz via JSON-RPC (eth_blockNumber, eth_getTransactionCount, eth_getBalance).
  2. Find last-active block via a bounded binary search over eth_getTransactionCount(atBlock) — up to 12 calls.
  3. Sample last 10 blocks for the wallet's recent txs (interaction diversity metric).
  4. Score 7 metrics: Recency (25%), Frequency (15%), Gas (10%), Dust (10%), Diversity (15%), Contract exposure (10%), Balance activity (15%).
  5. Combine with a weighted geometric mean — any 0-metric forces the overall to 0.

Install as an agent skill

Walletsleepscore is shipped as a Pharos Agent Center skill. Install it once and your agent can call the scorer as a tool.

  1. git clone https://github.com/kehindemariam/walletsleepscore
  2. Copy SKILL.md into your agent's skill directory (e.g. ~/.pharos/skills/).
  3. Ask your agent: "is this wallet healthy? 0xWALLET"

The agent will auto-load SKILL.md, run the bash or Python scorer, and return the same 0-100 report you see above — including the gauge, the 7 metrics, and the verdict label.
